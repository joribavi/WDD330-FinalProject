# INTRODUCTION
This is a Final Project for WDD-330 Course on BYUI Pathway Wordlwide

The page works with Github API and Open Library API (fetched from the searchbar on the main index file). 
Other pages run with json files save for Study Plan sub page , that works with js to create and delete tasks saved on local Storage. 

# REQUIREMENTS
Base knowledge of Javascript, Html, Tailwind Css
Have installed Octokit for fetching data from Github Rest API
Install vite and configure workflow at github pages if deployment is done on Github Pages 
Run npm run dev for local deployment (or add npm run start on github.config) 
Eslint is optional
